php bin/magento cache:flush: Flushes the cache storage, clearing all cached data.

php bin/magento setup:upgrade: Upgrades the Magento application and applies database schema and data changes, including module installation and updates.

php bin/magento setup:di:compile: Compiles the dependency injection configuration and generates the DI configuration files for faster performance.

php bin/magento indexer:reindex: Reindexes data for catalog search, prices, URL rewrites, and other indexers.

php bin/magento module:status: Checks the status of Magento modules (enabled or disabled).

php bin/magento module:enable Module_Name: Enables a specific Magento module.

php bin/magento module:disable Module_Name: Disables a specific Magento module.

php bin/magento cache:clean: Cleans the cache types specified in the Magento configuration.

php bin/magento cache:enable: Enables the cache types specified in the Magento configuration.

php bin/magento cache:disable: Disables the cache types specified in the Magento configuration.

php bin/magento deploy:mode:show: Shows the current deployment mode of Magento (e.g., default, developer, production).

php bin/magento deploy:mode:set Mode_Name: Sets the deployment mode of Magento (e.g., default, developer, production).
